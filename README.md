**Spam Detection using Machine Learning**
Spam detection is an important Natural Language Processing (NLP) task used to classify messages as Spam or Ham (Not Spam).
This project builds a machine learning model that automatically detects whether a message is spam based on its text content.
The goal is to reduce unwanted messages and improve communication filtering systems.
